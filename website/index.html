Built different

Not another terminal with an AI sidebar. Ocean is infrastructure for agent workflows.

Session Isolation

Each session gets a copy-on-write filesystem clone. Agents work in parallel without stepping on each other. Zero-cost, instant, APFS-native.

~50ms spawn · ~KB per session

Conflict Resolution

Real-time detection when sessions edit the same files. 3-way merge with AI-assisted resolution. Accept, edit, or let Claude merge it.

file + line-level detection

Ship to PR

Merge sessions, generate a commit, and open a pull request. Multiple agents, one clean PR. Git is the translation layer, not the workflow.

Cmd+Shift+S

Agent Awareness

Detects Claude, Codex, Aider, Cursor, Copilot, and more. Status badges, waiting notifications, and a dashboard for every running agent.

Ambient Awareness

System metrics, git status, network health, port detection — always visible in the status bar. Know what's happening without switching context.

~60MB idle memory
