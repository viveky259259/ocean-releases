Why Ocean

No terminal was built for orchestrating AI agents. Until now.

Zero-Cost Session Isolation

Copy-on-write filesystem clones via APFS. Each agent gets its own workspace — instant, costs kilobytes. No more 2GB git worktrees or 3-minute setup.

~50ms spawn · ~50KB per session

Real-Time Conflict Detection

Know the moment two agents edit the same file. Severity classified: safe (auto-merge), overlapping (review), conflicting (manual). No more surprise merge hell.

file + line-level detection

AI-Assisted Merge

3-way merge engine with per-hunk controls. Accept A, B, Both, Base, or let Claude merge intelligently with confidence scoring. Merge queue recommends optimal order.

Cmd+Shift+M

Agent Detection & Dashboard

Auto-detects Claude Code, Codex, Aider, Cursor, Copilot, Cody, Gemini, Devin. Status badges on panes, waiting alerts after 30s, full dashboard view.

8 agents supported

Ship to PR in One Command

Session DAG to git commits to pull request. Agent metadata in commit trailers for full attribution. Multiple agents, one clean PR.

Cmd+Shift+S

Interactive Terminal Output

Every file path, URL, error, and stack trace is clickable. Right-click for context actions. Annotation engine runs at <0.5ms per line.

clickable annotations

Ambient Awareness

Status bar shows CPU, RAM, network, git branch, ahead/behind, staged files, active agents — always visible. Port detection with one-click public tunneling.

real-time metrics

Native & Lightweight

Tauri v2 (Rust) + SolidJS + xterm.js WebGL. Not Electron. Launches in 400ms, idles at 60MB. macOS-native, keyboard-first, open-source.
